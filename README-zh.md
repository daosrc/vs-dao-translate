# VS DAO Translate

一个强大的翻译插件，支持划词翻译和悬停翻译功能。

## 功能

- 悬停翻译：将鼠标悬停在单词或短语上即可查看翻译
- 划词翻译：选择文本后自动翻译或通过命令翻译
- 支持多种翻译服务（Google翻译、本地词典）
- 内置常用词汇词典
- 可自定义设置

## 设置

- `vs-dao-translate.translationService`: 选择翻译服务提供者
- `vs-dao-translate.targetLanguage`: 目标翻译语言
- `vs-dao-translate.sourceLanguage`: 源语言（设为auto自动检测）
- `vs-dao-translate.enableHoverTranslation`: 启用悬停翻译功能
- `vs-dao-translate.enableSelectionTranslation`: 启用划词翻译功能

## 使用方法

1. 选择文本后右键选择"Translate Selected Text"
2. 或使用快捷键触发翻译命令
3. 悬停在单词上查看翻译结果
