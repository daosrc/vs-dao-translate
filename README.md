# VS DAO Translate

A powerful translation plugin that supports selection translation and hover translation features.

## Features

- Hover translation: View translations by hovering over words or phrases
- Selection translation: Automatically translate selected text or translate via command
- Support for multiple translation services (Google Translate, Local Dictionary)
- Built-in common vocabulary dictionary
- Customizable settings

## Settings

- `vs-dao-translate.translationService`: Select translation service provider
- `vs-dao-translate.targetLanguage`: Target translation language
- `vs-dao-translate.sourceLanguage`: Source language (set to auto for automatic detection)
- `vs-dao-translate.enableHoverTranslation`: Enable hover translation feature
- `vs-dao-translate.enableSelectionTranslation`: Enable selection translation feature

## Usage

1. Select text and right-click to choose "Translate Selected Text"
2. Or use keyboard shortcuts to trigger translation command
3. Hover over words to view translations
